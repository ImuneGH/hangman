# **Hangman**  
**Zahraj si zde:** [https://hangman-cz.vercel.app/](https://hangman-cz.vercel.app/)

### **CZ Popis**  
Hangman je český oběšenec, tedy hra, kde hráč hádá písmena, které mají dát dohromady nějaké slovo. Pokud hráč trefí písmeno, které je součástí hádaného slova, doplní se do něj a poodhalí tím tajenku, pokud ne, postaví se kus šibenice. Když se hráči podaří uhádnot celé slovo, vyhrál, když se ale dřív dokončí šibenice, prohrál.
Pojď si zahrát na odkazu výše, v mojí verzi oběšence najdeš možnost volby tématu i obtížnosti. Obtížnost určuje pouze počet pokusů před dokončením šibenice.

### **EN Description**  
Hangman is a classic word-guessing game. The player tries to guess a hidden word by suggesting letters one at a time. If the chosen letter is part of the word, it is revealed in its correct position. If not, a part of the gallows is drawn. The player wins by guessing the entire word before the drawing of the gallows is completed. Otherwise, the game is lost.
Play the game using the link above. In my version of Hangman, you can choose a theme and difficulty. The difficulty only affects the number of attempts before the gallows is completed.
